# AI CSP Solver

An AI-based Constraint Satisfaction Problem (CSP) Solver built in Python.

## Features

- Generic CSP framework
- Multiple solving strategies:
  - Basic Backtracking
  - Minimum Remaining Values (MRV)
  - Least Constraining Value (LCV)
  - Forward Checking
- Interactive Tkinter GUI
- Performance comparison charts
- Two implemented problems:
  - Magic Square (3x3)
  - University Exam Scheduling

## Project Files

- `main.py` - Launches GUI
- `gui.py` - User Interface
- `csp.py` - CSP model
- `solver.py` - Search algorithms
- `magic_square.py` - Magic Square problem
- `exam_scheduling.py` - Exam Scheduling problem
- `experiments.py` - Charts / comparisons
